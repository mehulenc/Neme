# Vision

This project is my personal financial operating system.

The long-term goal is to bring together the major parts of my financial life in one private, trustworthy place so I can understand, control, and improve it over time.

## What the system should eventually help me do

- Reconcile shared expenses between my bank/card activity and Splitwise.
- See where my money is going and where I overspend.
- Understand recurring spending patterns and avoidable costs.
- Track investments and evaluate whether they are aligned with long-term goals.
- Estimate whether current saving and investing behavior supports the kind of lifestyle I want later.
- Use AI-assisted analysis to surface useful insights from my financial history.

## Product philosophy

- The system must be trustworthy before it is clever.
- Financial data should be modeled explicitly, with clear provenance and auditability.
- Reconciliation and correctness matter more than visual polish.
- AI is a support layer, not the source of truth.
- The system should start narrow and become broader only after the core workflow is stable.

## Core idea

All financial activity should eventually live in one coherent system:

- bank and card transactions
- Splitwise expenses
- tags and categories
- investments
- derived insights

The first version should establish the foundation for that future system without trying to build everything at once.

## Near-term priority

The first useful version must make Splitwise reconciliation fast, accurate, and low-friction.
That is the first major workflow this product must solve well.
