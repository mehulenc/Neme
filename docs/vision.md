# Vision

Neme is a personal financial operating system.

The long-term goal is to bring together the major parts of your financial life in one private, trustworthy place so you can understand, control, and improve it over time.

## Eventual Capabilities

- Reconcile shared expenses between bank/card activity and Splitwise.
- See where money is going and identify overspending.
- Understand recurring spending patterns and avoidable costs.
- Track investments and evaluate alignment with long-term goals.
- Estimate whether current saving and investing behavior supports future lifestyle goals.
- Use AI-assisted analysis to surface useful insights from financial history.

## Product Philosophy

- The system must be trustworthy before it is clever.
- Financial data should be modeled explicitly, with clear provenance and auditability.
- Reconciliation and correctness matter more than visual polish.
- AI is a support layer, not the source of truth.
- The system should start narrow and become broader only after the core workflow is stable.

## Core Idea

All financial activity should eventually live in one coherent system:

- Bank and card transactions
- Splitwise expenses
- Tags and categories
- Investments
- Derived insights

The first version establishes the foundation for that future system without trying to build everything at once.

## Near-Term Priority

The first useful version must make Splitwise reconciliation fast, accurate, and low-friction. That is the first major workflow this product must solve well.
